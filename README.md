# Express sample

Simple GET/POST with express to edit a json file

## Getting Started
Install via below 
```
npm install 
or
yarn install
```

### Prerequisites

Needs node > 6 
Postman to test 

### To run app
```
npm run start
or
yarn run start
```

### Get URLS
 - http://localhost:4000/gods
 - http://localhost:4000/god/:godId

### Test with Postman

URL : http://localhost:4000/god/add
HTTP verb : POST
Body : x-www-form-urlencoded
keys : name , tag

## Built With

* Express js

### Misc 
Uses prettier to format. To Format 
```
npm run format
or
yarn run format
``` 
